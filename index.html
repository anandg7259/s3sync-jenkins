Get started

Welcome

Download this file as index.html, upload it into the root of your S3 bucket, enable Static website hosting, make objects public (or use CloudFront), and visit the website endpoint.

Why this works

  • All content is static HTML/CSS/JS — no server required.
  • Objects in S3 will be served as regular web assets (set index.html as index document).
  • For HTTPS and custom domains, add CloudFront and a certificate.
